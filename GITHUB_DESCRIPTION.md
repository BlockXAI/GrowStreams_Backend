# 🌊 GrowStreams Backend

**Real-time money streaming protocol on Vara Network with AI-powered contribution rewards**

Backend infrastructure for GrowStreams — a per-second token streaming platform built on Vara Network. Features include:

- 💸 **Continuous Payment Streams** — Per-second token flows for payroll, subscriptions, and grants
- 🎯 **AI-Powered Campaign System** — GPT-4o-mini scoring for OSS contributions and content creation
- 🏆 **Automated XP & Rewards** — Daily accumulation, merge bonuses, and USDC payouts
- 🔗 **REST API** — Complete API for streams, vault, tokens, and leaderboards
- 🤖 **Smart Contracts** — Rust-based Vara contracts (StreamCore, TokenVault, GROW token)
- 🔄 **GitHub & Twitter Integration** — Webhook-driven contribution tracking with anti-gaming measures

**Tech Stack:** Express.js • Vara Network • Rust/Sails • Supabase • OpenAI • Railway
