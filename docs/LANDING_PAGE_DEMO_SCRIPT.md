## 🎥 Production Notes

### Timing Breakdown
- **Section 1:** Hero & Value Prop (0:30)
- **Section 2:** Wallet Connection (0:30)
- **Section 3:** Identity Linking (0:40)
- **Section 4:** Dashboard Preview (0:30)
- **Section 5:** CTA & Close (0:20)
- **Total:** 2:30 minutes

### Visual Style
- **Clean & Modern:** Minimal UI, lots of white space
- **Smooth Transitions:** Fade, slide, no jarring cuts
- **Hover Effects:** Show interactive elements clearly
- **Highlight Spotlights:** Draw attention to key areas
- **Color Scheme:** Blues & greens (water/streaming theme)

### Key UI Elements to Show
1. ✅ Hero section with live stats
2. ✅ Wallet connection modal
3. ✅ GitHub/Twitter OAuth flows
4. ✅ Profile completion screen
5. ✅ Clean dashboard layout
6. ✅ Empty state messaging
7. ✅ Quick action buttons

### Recording Tips
1. **Use a clean test account** - 0 streams, fresh profile
2. **Pre-record OAuth screens** - GitHub/Twitter popups may not record well
3. **Slow cursor movements** - Give viewers time to read
4. **Pause on key screens** - 2-3 seconds minimum
5. **Use screen highlights/spotlights** - Focus viewer attention

### Voiceover Style
- **Tone:** Friendly, welcoming, not overly technical
- **Pace:** Moderate (slightly slower for UI navigation)
- **Energy:** Medium-high, enthusiastic but professional

### Music
- **Throughout:** Upbeat, modern, light electronic
- **Volume:** Background only, don't overpower voiceover
- **Style:** Something like "corporate uplifting" or "tech startup"

### Text Overlays
- Minimal - let the UI speak for itself
- Use only for:
  - Section transitions
  - Key callouts (e.g., "✨ Link Identity = Build Trust")
  - Final CTA

### For Twitter/X (60 seconds)
Cut down to:
1. Hero (10s)
2. Connect wallet (15s)
3. Link GitHub/Twitter (15s)
4. Dashboard quick look (15s)
5. CTA (5s)

### For YouTube Shorts / TikTok (30 seconds)
Ultra-condensed:
1. Problem hook: "Tired of delayed payments?" (5s)
2. Solution: "GrowStreams = real-time streaming" (5s)
3. Quick connect wallet (8s)
4. Show dashboard (7s)
5. CTA: "Try now, link in bio" (5s)

### For Onboarding Tutorial (Inside App)
Make it interactive:
- Pause after each step
- Add "Click here to continue" prompts
- Include tooltips and helper text
- Allow user to actually complete actions
